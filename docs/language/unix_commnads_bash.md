### Samples 

Remove files under folder
```
rm -rf /path/to/directory/*
```

Search process by keyword
```
ps -ef | grep {keyword}
```
Search file by keyword
```
ls {folder path} | grep {keyword}
```

Search log info by keyword
```
cat {log file path} | grep {keyword}
```
