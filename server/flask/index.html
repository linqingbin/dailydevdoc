    Flask¶

    Official Doc

    Start app¶

    from flask import Flask
app = Flask(__name__)

@app.route("/")
def home():
    return "Hello FLask"

app.run(host='0.0.0.0',port=5000,debug=True)

    Often imported modules

    from flask import (
    flash, g, redirect, render_template, request, session, url_for
)

    Route¶

    Methods: GET, POST, PUT, DELETE, HEAD, OPTIONS

    @app.route('/login', methods=['GET', 'POST'])
    • general router: @app.route(url)
    • before general router: @app.before_request
    • after general router: @app.after_request
    • before reuqest finished: @app.teardown_request

    Response types:

    • page: render_template('index.html')
    • redirect: redirect('/')
    • information: flash(“xxx”)
    • error: abort(404)

    Route with paramter

    @app.route('user/<name>')
def user(name):
    return '<h1>hello,%s!</h1>' % name
